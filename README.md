# LifeCO Cloud Connect

This repository contains the LIFECO Cloud Connect plant management dashboard built with React, Tailwind CSS, TanStack Router, Supabase integration, and shadcn/ui components.

## Repository

Remote origin set to:

`https://github.com/mohammedjad91-cell/lifeco-cloud-connect.git`

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the app in your browser at `http://localhost:5173`.

## Features

- Dashboards for plant operations, lab readings, and BI analytics
- Shift report generation and PDF export
- WhatsApp sharing from generated reports
- OST simulator and configurable OST charts
- Supabase-backed logging and PDF uploads
